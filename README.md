# Twitter-Sentiment-Analysis

## Project Overview

This project focuses on **Sentiment Analysis** using **Natural Language Processing (NLP)** and **Machine Learning** techniques to classify text sentiments into positive and negative categories.

The project was built using a large Twitter dataset containing over **1.6 million tweets**, where extensive text preprocessing and data cleaning techniques were applied to improve model performance and prediction accuracy.

The workflow includes:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Text Vectorization
* Machine Learning Model Building
* Model Evaluation

---

# Dataset Information

The dataset contains Twitter text data with sentiment labels.

### Features:

* Tweet Text
* Sentiment Label
